# Portafolio Personal

Este proyecto consiste en el desarrollo de una Landing Page de Portafolio Personal, realizada como la primera Práctica Formativa Obligatoria (PFO1) para la tecnicatura en Desarrollo de Software. El objetivo es aplicar conocimientos de maquetación con HTML5 y diseño responsivo con CSS3, integrando herramientas como Google Fonts y metodologías de despliegue continuo mediante GitHub Pages.
